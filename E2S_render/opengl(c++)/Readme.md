# Installation instructions
## Libraries requirements：
You can use `vcpkg` to install these Libraries.
## Get project：
 Please open `cmd` or `git bash` in the folder, then use `cmake .` to generate the c++ project of simulator where you can design scenes with OpenGL. Then, you can use visual c++ to start the project
 . 
## Version requirements：
 we recommend to use visual 2015(or later).


## multi scale sampling
you need to change model.h, vidar.h and main.cpp.
